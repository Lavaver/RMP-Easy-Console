# RMP Easy Console - 一款初中学生开发的 JSON 管理软件

其实这个软件当初是想要作为 RMP 配套开发的，但鉴于做到 JSON 这种地步了，也没理由不把它完善一下了（）

这款软件主要有三大功能：

- 基础的可分类 JSON 表格输出
- 基本的日志输出功能
- 基本的 JSON 新建、编辑一体引导式功能（*但不包括基本的打开编辑功能*）

然后就没有然后了（逃）

# 基础用法

其实这款软件把使用方法详细地说了一遍，我这里就简单赘述几句

## 首先是解析、查阅 JSON 文件结构的内容

也是**程序在默认情况下启动时所启动的内容**，一般情况下只需要将 JSON 文件拖入到控制台窗口中回车就能解析输出所有内容。

当然你也可以在启动之前就在控制台输入 `-easyhelp` 参数来看简易用法

## 基本的新建 JSON 与编辑功能

我认为这是我整个程序生涯中最有突破性的一个参数功能（喜）

使用 `-newjson` 参数启动程序，你会看到这样的界面：

```bash
快速添加 JSON
----------------------------
[1] 添加新的 JSON 文件至桌面并按引导写入值
[2] 添加新的 JSON 文件至其他位置并按引导写入值
[3] 仅在桌面添加空 JSON 文件
[4] 仅在其他位置添加空 JSON 文件
[5] 离开程序
请选择选项（1-5）：
```

这个功能很好理解也很好用，只是有点费眼睛。你只需要学习几分钟就可以迅速用正确的语法格式写完一个 JSON 文件，方便吧？
